# EntreSkill Hub 🚀
A Skill-to-Startup Enablement Platform for aspiring micro-entrepreneurs.

## Live Demo
- Frontend: https://entreskillhub-47oh.vercel.app
- Backend: https://entreskillhub.onrender.com

## Features
- User Registration & Login with JWT Auth
- Skill & Interest Profiling
- Business Idea Recommendations (15+ ideas)
- Step-by-step Business Roadmaps
- Progress Tracking & Bookmarks
- Mentor Directory
- Admin Dashboard

## Tech Stack
- Frontend: React.js, React Router, Axios
- Backend: Node.js, Express.js
- Database: MongoDB Atlas
- Deployment: Vercel (Frontend) + Render (Backend)

## Setup Instructions
### Backend
cd server
npm install
npm run dev

### Frontend
cd client
npm install
npm start
